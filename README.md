# PF1e Weapon Action Summary

A small Foundry VTT module for the Pathfinder 1e system.

It displays each weapon action’s damage in the weapon item sheet sidebar, above Quantity.

Example:

- Melee — 1d4 / P,S
- Throw — 1d4 / P,S

## Requirements

- Foundry VTT v13
- Pathfinder 1e system

## Notes

This module is visual-only. It does not modify items, actors, attacks, damage rolls, or system data.
